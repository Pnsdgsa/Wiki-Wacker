# **App Name**: Wiki Wacker

## Core Features:

- URL Input: Allow the user to input a URL for a Wiki Fandom page.
- HTML Parsing: Extract the main content from the Wiki Fandom HTML.
- Content Display: Display the extracted content in a readable format on the page.
- Image Handling: Download and display images from the Wiki Fandom page.
- Formatting Preservation: Attempt to preserve basic formatting (headings, lists, etc.) from the original page.

## Style Guidelines:

- Primary color: Soft blue (#A7D1FF) to evoke a sense of clarity and calmness, similar to informational websites.
- Background color: Very light grey (#F2F2F2), providing a neutral backdrop that doesn't distract from the content.
- Accent color: Muted purple (#B19CD9) for links and interactive elements, providing a subtle contrast.
- Font: 'Inter', a sans-serif font known for its readability and clean appearance; appropriate for body text and headlines.
- Simple, outline-style icons for navigation and actions.
- Clean, single-column layout with clear separation of content sections.
- Subtle transitions when loading and displaying content.
